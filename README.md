# Dog Facts API v1 Replica (Node.js + Express)

A simplified replica of the Dog Facts API that returns dog facts in JSON format.

## Requirements Met
- Built with Node.js and Express.js
- Endpoint: `GET /facts`
- Query parameter: `number` (integer)
- If `number` is missing: returns all facts
- Basic error handling for invalid requests
- Includes test cases for normal and edge scenarios
